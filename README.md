# Fuel Savings Calculator

Python-script om te berekenen of het loont om verder te rijden voor goedkopere brandstof.

## Features
- Rekent extra rijafstand heen/terug
- Houdt rekening met verbruik (km/L)
- Bepaalt minimale rendabele tankhoeveelheid
- Toont besparing bij opgegeven liters
- Referentie = dichtstbijzijnde pomp

## Gebruik

python FuelSavingsCalculation.py 20
